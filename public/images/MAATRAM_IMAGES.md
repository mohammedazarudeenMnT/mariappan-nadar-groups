# MaatRaM Trust Images

## Required Images

### Hero Section

- **File**: `maatram-hero.jpg`
- **Dimensions**: 1200x524px (recommended)
- **Description**: Main hero image showing MaatRaM Trust's community impact, education, or healthcare initiatives
- **Alt Text**: "MaatRaM Trust Community Impact"

## Image Guidelines

1. **Quality**: Use high-resolution images (minimum 1200px width)
2. **Format**: JPG or WebP for photos, PNG for graphics with transparency
3. **Optimization**: Compress images to reduce file size while maintaining quality
4. **Content**: Images should reflect:
   - Educational activities
   - Healthcare initiatives
   - Community engagement
   - Student success stories
   - Medical camps and health checkups

## Suggested Image Sources

- Professional photography from MaatRaM Trust events
- Stock photos from Unsplash, Pexels (ensure proper licensing)
- Custom graphics created for the trust

## Accessibility

- All images must have descriptive alt text
- Ensure good contrast for text overlays
- Consider color-blind friendly palettes
